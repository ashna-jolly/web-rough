DNS
It stands for Domain Name Server, connects URL with IP Addresses.
TCP
It stands for Transition Control Protocol, is a connection Oriented Protocol.
HTTP
It stands for HyperText Transfer Protocol, runs on the top of TCP/IP Protocol for file exchange in World Wide Web.
FTP
It stands for File Trnsfer Protocol, transfer of files from one host to another.
UDP
It stands for User Datagram Protocol,is a connectionless Protocol.
